Validator (and more) for Nanopublications

This validator interface for nanopublications is provided by Tobias Kuhn. It is based on the nanopub-java library and uses features from trustyuri-java for generating trusty URIs. It is work in progress and might contain bugs. The source code can be found on GitHub.

Input

Add a trusty URI to this nanopublication and download it as: TriG, TriX, N-Quads, RDF/JSON

Convert

Download nanopub as: TriG, TriX, N-Quads, RDF/JSON
